Tutorial: Using the Drag-and-Drop Code Translator

This tutorial will guide you through using the drag-and-drop code translator to create line code for a robotics kit. Follow these steps to drag code blocks, drop them in sequence, and see the translated output.

Step 1: Click on the Code Translator

Click the website on your computer or tablet. You will see three columns:

  • Code Blocks: The left column where you can see draggable blocks that represent different commands for your robot.
  • Drop Area: The middle column where you can drag the blocks to build a sequence of actions.
  • Translated Code: The right column where the dropped blocks will be translated into line coding instructions.

Step 2: Drag a Block

Start by clicking on a block from the Code Blocks column. Hold the mouse button down and drag it towards the Drop Area column.

Step 3: Drop the Block

Once you have dragged a block over to the Drop Area, release the mouse button to drop it in place. Youâ€™ll see the block appear in the sequence you are building.

The translated code of line coding will appear in the Code Preview column. This is a representation of the blocks of code that you made in line coding.
